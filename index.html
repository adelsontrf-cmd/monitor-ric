Configurações

🤖 API Gemini (análise automática de PDFs)

Configure sua chave da Google Gemini API para analisar automaticamente o inteiro teor dos requerimentos e pré-preencher os campos Objeto e Resumo. Obter chave.

A chave é armazenada apenas no seu navegador (IndexedDB), nunca enviada a outros servidores além do próprio Google.

Termos de busca (regex)

Regex aplicada à ementa normalizada (minúsculas, sem acento). O padrão pega "Casa Civil" e variações, incluindo nomes da Ministra atual e do anterior.

Unidades para consulta (multi-select no detalhamento)

Lista de siglas usada nos checkboxes do detalhe do requerimento. Uma por linha.

Situações não-mapeadas detectadas

Strings de situação que aparecem nas APIs e ainda não foram mapeadas para uma das 6 categorias internas. Use para refinar o de-para.

Nenhuma situação não-mapeada registrada ainda.

Proxy CORS (opcional)

Use {url} onde a URL alvo deve ser inserida. Ex.: https://corsproxy.io/?url={url}.

Backup e dados

Dados ficam no navegador (IndexedDB). Faça backup periodicamente.
