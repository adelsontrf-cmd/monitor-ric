➕ Incluir requerimento manualmente

Informe a casa, o tipo e o número/ano. O sistema buscará na API oficial e, se encontrar, preencherá automaticamente ementa, autor, tramitação e links. Se não encontrar (a busca por número/ano é limitada no Senado), o requerimento é criado em branco para você completar manualmente.

☁ Sincronização na nuvem

Configuração

Sincronização single-user via GitHub Gist privado. Os dados são cifrados (AES-GCM 256) antes de sair deste computador. Informe um PAT fine-grained com permissão de Gists e o ID do Gist (ou crie um novo).

Ativar outro computador

Exporte a configuração neste computador e cole no outro para ativar a sincronização com um único paste.

Como configurar (passo a passo)

  1. No GitHub, gere um PAT fine-grained restrito a Gists (Account settings → Developer settings → Fine-grained tokens; em "Account permissions", habilite "Gists: Read and write").
  2. Cole o PAT no campo acima e clique em "+ Criar novo Gist" — o ID será preenchido automaticamente.
  3. Clique em Salvar configuração e depois em Sincronizar agora.
  4. Em cada computador adicional, use Exportar configuração aqui e Importar e ativar lá — um único paste resolve.
  5. Computador compartilhado? Clique em Desconectar ao terminar de usar, para remover as credenciais deste dispositivo.

Configurações

🤖 API Gemini (análise automática de PDFs)

Configure sua chave da Google Gemini API para analisar automaticamente o inteiro teor dos requerimentos e pré-preencher os campos Objeto e Resumo. Obter chave.

A chave é armazenada apenas no seu navegador (IndexedDB), nunca enviada a outros servidores além do próprio Google.

Termos de busca (regex)

Regex aplicada à ementa normalizada (minúsculas, sem acento). O padrão pega "Casa Civil" e variações, incluindo nomes da Ministra atual e do anterior.

Unidades para consulta (multi-select no detalhamento)

Lista de siglas usada nos checkboxes do detalhe do requerimento. Uma por linha.

Proxy CORS (opcional)

Use {url} onde a URL alvo deve ser inserida. Ex.: https://corsproxy.io/?url={url}.

Backup e dados

Dados ficam no navegador (IndexedDB). Faça backup periodicamente.

RICs ocultos e excluídos

Gerencie RICs que foram ocultados (preservados na base, mas fora de listas e estatísticas) ou excluídos (com snapshot para recuperação).

Feriados e dias não-úteis

O cálculo de prazos sugeridos das etapas leva em conta os feriados nacionais brasileiros (fixos e móveis: Carnaval, Sexta da Paixão, Corpus Christi). Adicione aqui feriados extras (ex.: pontos facultativos, recessos, feriados estaduais/distritais).
